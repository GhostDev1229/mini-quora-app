# Mini Quora Style Posts App

This is a simple **Node.js + Express + EJS** project where users can:
- ✍️ Create a post  
- 🔍 View post details  
- ✏️ Edit posts  
- ❌ Delete posts  

---

## 🚀 Features
- Implemented **CRUD operations** (Create, Read, Update, Delete)  
- Learned about **method-override** for supporting PATCH & DELETE in forms  
- Added **redirects** after updating/deleting posts for smooth user experience  
- Styled with **CSS**
